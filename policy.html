Information we store:

How we store:
Our information is fully hashed and encrypted in a database. This data is not accessible by anyone but the owner.

How long we store:
We store our data forever, if you would like a data deletion please join the support server.
Please keep in mind we have to store this data for commands to work as intended, if you are not happy with this do not use the bot or request your data be deleted.
